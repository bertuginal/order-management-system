## ‚ú® Features

‚ù∂ All customers are listed.
  
‚ù∑ New Customer is created.

‚ù∏ Lists customers with the letter 'C' in it.
  
‚ùπ Lists the total amount of invoices of customers registered in June.

‚ù∫ Lists all invoices in the system.
